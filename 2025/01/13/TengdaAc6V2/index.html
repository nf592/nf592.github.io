TengdaAc6V2

Vulnerability name

Tengda Router AC6V2.0RTL_V15.03.06.50 There is a binary command to inject vulnerabilities

Vulnerability

High risk

Vulnerability description

Tenda AC6V2.0 is a 1200M optical fiber -specific 11AC wireless dual -frequency router produced by Tenda (Shenzhen Auspicious Tengda Technology Co., Ltd.). This router has command injection vulnerabilities and can get system permissions by attackers.

Vulnerability details

Packaged through the Binwalk -ME, the vulnerability exists in the /bin /httpd file of the Squashfs -ROOT file system. This file is a TENDA AC6 responsible for Web request processing file.

Use the IDA reverse HTTPD file to find the DosystemCMD function, position the FormWriteFacmac function. Here we see that the MAC parameters transmitted by Websgetvar are assigned to the MAC variable name. In the absence, the MAC address is: 00: 01: 02: 11: 22: 22: 22:22: 33, there was no judgment on the MAC follow -up, and then the DosystemCMD function was performed directly. This function is the function of the system executing commands.

Check the formwriteFacmac function who is called

At the interface WriteFacmac, you can be sure that the existence of command injection
